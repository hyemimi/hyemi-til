### 일반적인 재귀 함수

```
function dfs(state) {
    if (조건에 도달) {
        // 결과 처리
        return;
    }

    for (가능한 선택지) {
        // 선택
        dfs(업데이트된 상태);
        // 선택 취소 (원복)
    }
}
```

- state = 현재 상태를 나타내는 변수들

## 오늘의 회고

내가 풀었던 문제에서는 조건에 해당되는 모든 선택지를 dfs를 통해 재귀 호출 하는데, 선택 취소는 따로 하지 않았다.

그 이유는 원본 배열을 건들지 않고 새로운 변수를 선언했기 때문이다.
